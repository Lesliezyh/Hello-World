> 切尔诺贝利核事故，或简称切尔诺贝利事件，<br>
> 是1986年4月26日于苏联乌克兰普里皮亚季市切尔诺贝利核电站发生的核反应堆破裂事故。<br>
> 该事故是历史上最严重的核电事故，也是首例被国际核事件分级表评为最高第七级事件的特大事故。<br>
