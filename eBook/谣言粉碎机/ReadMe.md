> 事實上，包括牙膏、洗面乳…等一些「軟管」包裝的產品，在包裝底部都標有各種顏色的色條，
> 該色條的正確名稱為「電眼定位點」，是軟管產品在生產過程中，提供機器對產品「封尾」時的定位與識別，
> 當產品裝填入軟管後，封尾機器上的電眼會感應色條，對軟管進行封管及裁剪的包裝作業。
> 而色條選色的基本概念，以與底色反差較大並且容易反光為主，
> 方便電眼進行感應，也因為黑色條標的感應效果最好，所以也最常見。
