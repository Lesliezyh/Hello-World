右、左、スワイプしよう

Tantanはあなたの現在地、距離、性別、好みによって付近にいる面白そうな人をリアルタイムに教えるよ。あなたはスワイプするだけ！右スワイプは好き、左スワイピはパス、超簡単でしょ〜

奇跡を見届けよう

お気に入りの人と両想いだったら最高だね！Tantanはマッチング成功のときだけ通知するので、パスしたことはバレないから心配無用！告白失敗？苦労して感動させても報われない？ここでは絶対起きない！

私の人生、私が選ぶ

Tantanでは、お互いが気に入った相手とだけチャットできるから、知らない人からの嫌がらせを受けることはない。うまくいかなくても、いつでも相手をブロックや通報できる。

このときめきを知ってるのは、お互いだけ
