关于探探“匿名暗恋短信”的说明

近日，一些手机号机主收到了探探App的“匿名暗恋表白”短信。由于缺乏相关信息和不了解App产品机制，令机主产生了一些误解。探探特此说明如下：

1. 匿名暗恋表白功能运作机制说明
探探App于2016年2月上线了匿名暗恋表白功能，旨在帮助互相暗恋彼此的用户，了解彼此心意，促成一段美好的缘分。
用户使用该功能时，会主动选择自己通讯录中他/她 所“暗恋”的联系人。选择后，系统会匿名推送一条短信到此手机号，帮助用户和“暗恋”的人进行表白。在这个过程中，用户的操作完全是自愿和匿名的，探探App不会主动帮用户操作。
被“暗恋”的人收到此短信后，可自行选择是否使用探探App。如果他/她选择使用App，并打开“匿名暗恋表白”功能，此人可以在探探上进行反向选择可能“暗恋”他/她的人。若猜中暗恋者，则双方配对成功，可以沟通、聊天；若没有猜中，“暗恋”者信息和被“暗恋”者的信息，也不会被暴露，双方无法知道彼此的身份。被“暗恋”的人的联系方式均来自于注册用户通讯录，不涉及买卖个人信息。

2. 全程安全传输，不触犯用户隐私
匿名暗恋表白功能属于用户自发的，一对一的定向操作，且操作过程均在app端完成，任何第三方无权也无法获知隐私信息。故此行为不会泄漏用户隐私信息，对用户信息安全不构成威胁。短信内容明确标注来自于【探探】，“姓名由手机联系人提供”，收到短信的机主可免费回T退订。

3. 严格限制用户使用，避免过度骚扰
此功能上线1年多的时间里，我们陆续从各个渠道接收到热心用户的反馈，一小撮用户频繁使用此功能进行恶作剧，利用匿名暗恋表白功能给一些手机机主发送了匿名暗恋短信。为此，探探会在产品功能上限制“恶作剧”用户的使用，避免过度骚扰，还手机机主一片短信的洁净“蓝天”！

4. 投诉邮箱：sms-report@tantanapp.com
如果您不想再收到此类短信，您可以通过邮件告知我们。邮件内容为收到短信的手机号+收到时间+收到的内容。我们会将手机号做妥善处理，保证不再向您发送此类短信。 或者您可以按照短信信息指引，直接免费回复T进行退订，也可以达到同样效果。
