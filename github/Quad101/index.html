  • Windows

    1. 點選開始列，並選擇進入「設定」項目。
    2. 選擇進入「網路和網際網路」。
    3. 選擇進入「變更介面卡選項」。
    4. 選擇您上網使用的介面設定項目，按右按鍵選擇「內容」項目。
    5. 選擇「網際網路通訊協定第4版(TCP/IPv4)」項目之後選「內容」進入設定，更改下方的「使用下列的DNS伺服器位址」，更改「慣用DNS伺服器」內容為「101.101.101.101」，之後按確定鍵完成設定。
    6. 再次新增另一DNS伺服器位址，輸入101.102.103.104，之後按確定鍵完成設定。
    7. 若要增加 IPv6的DNS設定，請選擇「網際網路通訊協定第6版(TCP/IPv6)」項目之後選「內容」進入設定，更改下方的「使用下列的DNS伺服器位址」，更改「慣用DNS伺服器」內容為「2001:de4::101」，之後按確定鍵完成設定。
    8. 再次按確定鍵輸入「2001:de4::102」，之後按確定鍵完成設定
    9. 您已完成設定! 享受更快及更安全的DNS服務吧!

  • MacOS

    1. 請先到「系統偏好設定」下的「網路」設定
    2. 選擇您正在連線的連線項目並按下「進階」選項.
    3. 此時會進入DNS設定畫面請以左下方的「-」號清除原本的DNS設定，之後以「+」號加入「101.101.101.101」，「101.102.103.104」，及IPv6 DNS Address 「2001:de4::101」及「2001:de4::102」
    4. 之後按「好」,回到上層視窗後按「套用」便完成設定.
    5. 您已完成設定! 享受更快及更安全的DNS服務吧!

  • iPhone

    1. 開啟 「設定」 選項。
    2. 點選 Wi-Fi項目,然後點選您正連線的WiFi設定檔。
    3. 下拉至DNS項目，將「自動」改為「手動」。
    4. Tap the - button, and Delete to clean any existing entries.
    5. 於列表中按項目前方的「-」號將原本的DNS設定清除。
    6. 點選「加入伺服器」之後便可以鍵入 101.101.101.101，101.102.103.104。
    7. 再點選「加入伺服器」之後便可以鍵入IPv6 DNS Address 2001:de4::101及2001:de4::102。
    8. 之後點選右上方「儲存」鍵。
    9. 您已完成設定! 享受更快及更安全的DNS服務吧!

  • Android

    1. 選擇WiFi項目,可見到列出的WiFi設定檔。
    2. 長按您正在連線的設定檔，選擇「更改網路設定」選項。
    3. 選擇「顯示進階選項」，之後選擇「DHCP」項目，將其改為「靜態」設定.
    4. 之後視窗下拉至最後便可見到DNS1 及 DNS2 設定，於DNS1設定處鍵入「101.101.101.101」，若您要增加IPv6 DNS Address，此處請鍵入「2001:de4::101」。
    5. 之後按下「儲存」。
    6. 您已完成設定! 享受更快及更安全的DNS服務吧!

Quad101 DNS 計畫

Quad101 是TWNIC推動的DNS解析服務實驗計畫。
TWNIC 台灣網路資訊中心為國家頂級網域名稱註冊管理機構(ccTLD: country code Top Level
Domain)，營運之DNS網路遍及全球不同區域。Quad101 DNS計劃導入 IP 位址
(101.101.101.101)，輔以TWNIC 優異的DNS全球網路，協助台灣邁向優質化的網際網路。
