  • Windows

    1. Click on the Start menu, then click on Control Panel.
    2. Click on Network and Internet.
    3. Click on Change Adapter Settings.
    4. Right click on the Wi-Fi network or Ethernet interface you are connected to, then click Properties.
    5. Select Internet Protocol Version 4.If you want setup IPv6 DNS server address,select Internet Protocol Version 6
    6. Click Properties.
    7. Write down any existing DNS server entries for future reference.
    8. Click Use The Following DNS Server Addresses.
    9. Replace those addresses with the 101.101.101.101 and 101.102.103.104 DNS addresses,IPv6 DNS Server Address please type 2001:de4::101 and 2001:de4::102.
    10. Click OK, then Close.
    11. You’re all set! Enjoy more faster and private DNS servers.

  • MacOS

    1. Open System Preferences.
    2. Search for DNS Servers item and select it.
    3. Click the + button to add a DNS Server and enter 101.101.101.101 and 101.102.103.104,you can also add a IPv6 DNS Server address enter 2001:de4::101 and 2001:de4::102.
    4. Click Ok, then click Apply.
    5. You’re all set! Enjoy more faster and private DNS servers.

  • iPhone

    1. Open the Settings app.
    2. Tap Wi-Fi, then tap your preferred network in the list.
    3. Tap Configure DNS, then tap Manual.
    4. Tap the - button, and Delete to clean any existing entries.
    5. Tap the + Add Server button, then type 101.101.101.101
    6. Tap the + Add Server button, then type 101.102.103.104
    7. Tap the + Add Server button, then type 2001:de4::101
    8. Tap the + Add Server button, then type 2001:de4::102
    9. Tap the Save button on the top right.
    10. You’re all set! Enjoy more faster and private DNS servers.

  • Android

    1. Go Settings than tap WiFi
    2. Long press the Network you're prefer connected to, and tap 'Modify Network'.
    3. Check 'Advanced'.
      Switch the IP settings from DHCP to 「Static」.
      Add 101.101.101.101 DNS servers to the entries for DNS 1. You can also add v6 DNS address 2001:de4::101 to the entries.
    4. Click 'Save'
    5. You’re all set! Enjoy more faster and private DNS servers.

Quad101 DNS Project

Quad101 is TWNIC's experimental Public DNS project. TWNIC (Taiwan Network Information Center) is a ccTLD (country-code Top Level Domain) registry operator. TWNIC runs one of the world's fastest DNS infrastructure.
The specific IP address (101.101.101.101) has been assigned to Quad101 Public DNS systems. TWNIC's world-class network infrastructure along with comprehensive Quad101 features will build a better Internet in Taiwan.
