以互联网为基础的科技和文化公司

腾讯，1998年11月诞生于中国深圳，是一家以互联网为基础的科技与文化公司。

我们的使命是 “通过互联网服务提升人类生活品质”。

腾讯秉承着 “一切以用户价值为依归” 的经营理念，致力于为亿万网民提供优质的互联网综合服务。
