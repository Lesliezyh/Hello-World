- [7-Zip](https://www.7-zip.org/)

```
7z 是一种主流高效的压缩格式，它拥有极高的压缩比。

在计算机科学中，7z是一种可以使用多种压缩算法进行数据压缩的档案格式。

该格式最初被7-Zip实现并采用，但是这种档案格式是公有的，并且7-Zip软件本身亦在GNU宽通用公共许可证(GNU LGPL)协议下开放源代码。

目前LZMA软件开发工具包的最新版本为V9.34。7z格式的MIME类型为application/x-7z-compressed。

通常使用7-Zip的 7z 格式能比使用 zip 格式的压缩文件小 30-70%。

并且使用7-Zip创建的 zip 格式比大多数其它压缩软件创建的都小 2-10%。
```
