【Tips】 PPT 小技巧 ：

- [ppt内如何插入动态时间](https://jingyan.baidu.com/article/af9f5a2d2bff2843150a4555.html)_百度经验

> **重点**：1.插入文本框。2.插入--》日期和时间--》勾选：“保持更新”选项。
>
> (**在ppt中实时显示时间**。具体做法如下： 点击“视图”中的“页眉和页脚”，在“幻灯片”对话框中选中“日期和时间”以及“自动更新选择”，在下拉菜单中可以选择显示时间的格式。)
